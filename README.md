# Thinking Machines Website

This repository contains the source code for the Thinking Machines Lab static website.

## Deployment with GitHub Pages

This site is designed to be hosted directly from your GitHub repository using GitHub Pages. Follow these simple steps to deploy it:

1.  **Navigate to Repository Settings:** In your repository, click on the **Settings** tab.
2.  **Go to Pages:** In the left-hand sidebar, click on **Pages**.
3.  **Select Deployment Source:** Under the "Build and deployment" section, select **Deploy from a branch** as your source.
4.  **Choose the Branch:** Select the `main` (or `master`) branch and keep the folder as `/root`.
5.  **Save Changes:** Click **Save**.

That's it! GitHub will build and deploy your site. It may take a few minutes, but your live site will be available at `https://<your-username>.github.io/<your-repository-name>/`.
